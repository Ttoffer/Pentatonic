# Pentatonic Scales (guitar)

Interactive **major / minor pentatonic** reference: a **circle of fifths** to pick the key and a **six-string fretboard** with pattern overlays, CAGED windows, and optional degree labels.

## Live site (GitHub Pages)

With **GitHub Pages** enabled for this repo, your app is served at the URL shown under **Settings → Pages** (for example `https://YOUR_USERNAME.github.io/YOUR_REPO/`).

## iPhone and iPad — use Safari

For the best experience and for adding an icon to your home screen, **use Safari**, not Chrome or another browser.

1. Open **Safari** on your iPhone or iPad.
2. Go to your **GitHub Pages** address (the `https://…github.io/…` link).
3. Tap the **Share** button (square with an arrow pointing up).
4. Scroll and tap **Add to Home Screen**.
5. Edit the name if you like (e.g. “Pentatonic”), then tap **Add**.

You’ll get an icon that opens the app **without the Safari address bar**, similar to a small installed app. Repeat on each device you use.

**Tip:** If you change the site later and the icon or title looks old, remove the old home screen shortcut and add it again from Safari.

## Requirements

None. Static files only — any static host or local file open in a browser works. **HTTPS** (GitHub Pages provides this) plus **Safari** is what you need for **Add to Home Screen** on Apple devices.

## Privacy

Runs entirely in the browser. No accounts, analytics, or server calls are included in this project.
