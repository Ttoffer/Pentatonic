# Pentatonic Scales (guitar)

Interactive **major / minor pentatonic** reference: a **circle of fifths** to pick the key and a **six-string fretboard** with pattern overlays, CAGED windows, and optional degree labels.

## Live site (GitHub Pages)

After Pages is enabled in the repo settings, open the published URL in **Safari** on iPhone or iPad, then **Share → Add to Home Screen** for an app-style icon.

## Files

| File | Role |
|------|------|
| `index.html` | Single-page app (HTML, CSS, and JavaScript) |
| `manifest.webmanifest` | Web app metadata (name, theme, standalone display) |
| `apple-touch-icon.png` | Home screen icon (iPhone / iPad) |

## Requirements

None. Static files only — any static host or local file open in a browser works; **HTTPS** + Safari is best for **Add to Home Screen**.

## Privacy

Runs entirely in the browser. No accounts, analytics, or server calls are included in this project.
